GR Glam Salon Web

GR Glam Salon Web es una página web desarrollada para un salón de belleza, creada con el objetivo de ofrecer una presencia digital moderna y facilitar la interacción con los clientes.

Sitio Web
https://lifesys.github.io/gr-glam-salon-web/

Estado del Proyecto
Actualmente el proyecto se encuentra en desarrollo. La versión actual permite a los usuarios agendar citas de manera sencilla.

Funcionalidades Actuales
- Información general del salón.
- Diseño responsivo para diferentes dispositivos.
- Formulario para agendar citas.
- Interfaz moderna y fácil de usar.

Mejoras Planificadas
Las siguientes funcionalidades se encuentran en proceso de implementación:

- Catálogo de servicios.
- Galería de trabajos realizados.
- Gestión de citas en línea.
- Información detallada de promociones.
- Integración con redes sociales.
- Panel administrativo para gestión de contenido.

Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap

Objetivo
Desarrollar una plataforma web que permita al salón de belleza mostrar sus servicios, mejorar la atención al cliente y facilitar la reserva de citas de manera digital.
